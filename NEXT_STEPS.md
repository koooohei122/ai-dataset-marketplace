# 次のステップ：サイト開設に向けて

プロジェクトの基本構造は準備できました！次に進むために、以下の情報を教えてください。

## 📋 必要な情報

### 1. アカウント情報（必須）

以下のアカウントを作成してください：

- [ ] **GitHubアカウント**
  - URL: https://github.com
  - 用途: コード管理
  - 作成後、ユーザー名を教えてください

- [ ] **Vercelアカウント**
  - URL: https://vercel.com
  - GitHubアカウントでログイン可能
  - 作成後、確認してください

- [ ] **Supabaseアカウント**
  - URL: https://supabase.com
  - GitHubアカウントでログイン可能
  - プロジェクト作成後、以下を教えてください：
    - Project URL
    - Database URL
    - anon key
    - service_role key

### 2. サイト情報（オプション）

- **サイト名**: AIデータセットマーケットプレイス（変更可能）
- **説明**: AI開発者、研究者、企業向けのデータセット販売・購入プラットフォーム
- **デフォルト言語**: 日本語

### 3. 機能の優先順位

どの機能から実装しますか？

1. **認証システム**（ログイン・登録）
2. **データセット一覧・検索**
3. **データセット詳細表示**
4. **購入フロー**
5. **ファイルアップロード**

## 🚀 今すぐできること

### ステップ1: 依存関係のインストール

```bash
cd ai-dataset-marketplace
pnpm install
# または
npm install
```

### ステップ2: 開発サーバーの起動

```bash
pnpm dev
# または
npm run dev
```

ブラウザで `http://localhost:3000` にアクセスして、動作確認してください。

### ステップ3: サービスアカウントの作成

上記のアカウントを作成してください。作成方法は `SETUP_INSTRUCTIONS.md` を参照してください。
