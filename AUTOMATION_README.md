# AI自動化開発基盤

要件を入力するだけで、仕様書からコード、テスト、レビュー、デプロイまで自動化するシステムです。

## 🎯 機能

1. **要件入力**: 自然言語でシステムの要件を記述
2. **仕様書生成**: AIが要件から詳細な技術仕様書を自動生成
3. **コード生成**: 仕様書から完全なアプリケーションコードを自動生成
4. **テスト生成・実行**: テストコードを自動生成して実行
5. **コードレビュー**: AIがコードの品質、セキュリティ、パフォーマンスをレビュー
6. **自動デプロイ**: 完成したシステムを自動でデプロイ

## 🚀 使い方

1. `/automation` ページにアクセス
2. 作りたいシステムの要件をテキストエリアに入力
3. 「自動開発を開始」ボタンをクリック
4. 進捗をリアルタイムで確認
5. 完了後、デプロイされたURLにアクセス

## 📋 要件入力の例

```
ユーザー認証機能付きのタスク管理アプリを作成してください。
- ログイン・ログアウト機能
- タスクの追加・編集・削除
- タスクの完了状態の管理
- タスクのフィルタリング（全て/未完了/完了）
- レスポンシブデザイン対応
```

## 🛠️ 技術スタック

- **フロントエンド**: Next.js 14 (App Router), TypeScript, Tailwind CSS
- **バックエンド**: Next.js API Routes
- **AI**: OpenAI GPT-4 (または Anthropic Claude)
- **デプロイ**: Vercel API

## ⚙️ セットアップ

### 環境変数

`.env.local` ファイルに以下を追加:

```env
# OpenAI API (必須)
OPENAI_API_KEY=sk-...

# Vercel API (デプロイ機能を使用する場合)
VERCEL_TOKEN=...

# または Anthropic API を使用する場合
ANTHROPIC_API_KEY=...
```

### インストール

```bash
pnpm install
pnpm dev
```

## 🔄 処理フロー

```
要件入力
  ↓
仕様書生成 (AI)
  ↓
コード生成 (AI)
  ↓
テスト生成・実行 (AI)
  ↓
コードレビュー (AI)
  ↓
問題があれば修正 (AI)
  ↓
デプロイ (Vercel API)
  ↓
完成したシステム
```

## 📊 進捗管理

各ステップの進捗はリアルタイムで更新されます:

- **仕様書生成中**: 10-30%
- **コード生成中**: 30-60%
- **テスト実行中**: 60-80%
- **レビュー中**: 80-90%
- **デプロイ中**: 90-100%
- **完了**: 100%

## 🎨 カスタマイズ

### AIモデルの変更

`lib/ai/automation.ts` の `AutomationEngine` コンストラクタでモデルを指定:

```typescript
const automationEngine = new AutomationEngine({
  model: 'gpt-4', // または 'gpt-3.5-turbo', 'claude-3-opus'
  temperature: 0.7,
});
```

### デプロイ先の変更

`lib/automation/deploy.ts` の `DeployEngine` をカスタマイズして、他のプラットフォーム（Netlify、Railway等）に対応可能。

## ⚠️ 注意事項

- **APIキー**: OpenAI APIキーが必要です（有料）
- **デプロイ**: Vercelトークンが必要です（無料プランあり）
- **ストレージ**: 現在はメモリ内ストレージを使用。本番環境ではデータベースが必要
- **処理時間**: AI処理には時間がかかります（数分〜数十分）
