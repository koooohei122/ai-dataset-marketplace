# テスト用要件例

AI自動化基盤をテストするための要件例です。コピー＆ペーストして使用できます。

## 簡単なテスト（推奨）

```
Hello Worldを表示するシンプルなNext.jsページを作成してください。
```

## 中程度のテスト

```
ユーザー認証機能付きのタスク管理アプリを作成してください。
- ログイン・ログアウト機能
- タスクの追加・編集・削除
- タスクの完了状態の管理
- レスポンシブデザイン対応
```

## 複雑なテスト

```
ECサイトの商品一覧ページを作成してください。
- 商品の検索機能
- カテゴリフィルタ
- 価格ソート
- ページネーション
- 商品詳細モーダル
- カートへの追加機能
- Next.js 14 App Router + TypeScript + Tailwind CSSを使用
```

## 使用手順

1. http://localhost:3000/automation にアクセス
2. 上記の要件をコピー＆ペースト
3. 「自動開発を開始」をクリック
4. 進捗を確認

## 注意事項

- 最初のテストは簡単な要件から始めることをお勧めします
- 処理には数分〜数十分かかる場合があります
- APIキーが正しく設定されていることを確認してください
